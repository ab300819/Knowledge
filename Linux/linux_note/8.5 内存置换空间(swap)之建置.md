## **8.5. 内存置换空间(swap)之创建**

### **8.5.1 使用实体分区创建swap**

1. 分区
```
fdisk /dev/hdc
```

2. 建立swap格式
```
mkswap /dev/hdc7
```

3. 查看与加载
```
swapon /dev/hdc7
free
```

### **8.5.2 使用文件建置swap**
1. 使用 `dd` 这个命令来新增一个 128MB 的文件在 `/tmp` 底下
```
dd if=/dev/zero of=/tmp/swap bs=1M count=128
```

2. 使用 `mkswap` 将 `/tmp/swap` 这个文件格式化为 swap 的文件格式
```
mkswap /tmp/swap
```

3. 使用 `swapon` 来将 `/tmp/swap` 启动
```
swapon /tmp/swap
```

4. 使用 `swapoff` 关掉 swap file
```
swapoff /tmp/swap
swapoff /dev/hdc7
```
